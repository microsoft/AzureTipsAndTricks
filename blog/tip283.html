    # Azure CLI examples

    The Azure CLI is a great way to get things done in Azure. But it isn't always easy to know what you can do with the CLI and how to do that. Sure, you can look at the help instructions, but those don't provide hands-on examples.

    The Azure Kung Fu Toolkit is created by Chris Pietschmann who is the founder and primary author at Build5Nines.com. The Azure Kung Fu Toolkit contains tons of practical scripts and examples for the Azure CLI. In this post, we'll show how to get started with the toolkit.

    # Prerequisites

    If you want to follow along, you'll need the following:

    • An Azure subscription (If you don't have an Azure subscription, create a free account before you begin)
    • The Azure CLI. You can download it for Windows, Linux or Mac
    • Visual Studio Code
    • Alternatively, you can provision the Azure Kung Fu VM to get started

    # Getting started with the Azure Kung Fu Toolkit

    Let's take a lap around the Azure Kung Fu Toolkit. Let's start by getting it on our local machine:

    1. Go to the Azure Kung Fu Toolkit GitHub page https://github.com/Build5Nines/az-kung-fu

    (Azure Kung Fu Toolkit GitHub repository)

    1. Download the repository .zip file
    2. Unzip the .zip file
    3. Open the unzipped repository folder with Visual Studio Code

    Alternatively, you can also clone the Azure Kung Fu Toolkit repository using the VS Code Git Clone command.

    Once you have opened the toolkit in VS Code, you need to make sure that you have the VS Code Azure CLI Tools Extension installed. If you don't:

    1. In VS Code, go to the Extensions tab
    2. Search for Azure CLI Tools
    3. Install the Azure CLI Tools extension

    Now you can browse through the folders of the toolkit and see which scripts are available. There are scripts for creating and deleting Azure Resource Groups. Scripts for starting and stopping Azure Virtual Machines. Scripts for managing Azure Virtual Networks and lots more. There are also a lot of samples that you can use for specific scenarios, like alter managed disks in a resource group.

    You can easily run the scripts from VS Code by replacing the parameter values in a script with your own values, selecting the script and selecting Terminal > Run Selected Text. Make sure that your default VS Code terminal is set to Git Bash as all the scripts are written in Bash.

    (Run scripts in Visual Studio Code)

    Let's try it out by using the scripts to create an Azure Web App:

    1. Navigate to the group > create folder and select the group-create.azcli file. We'll use this to create an Azure Resource Group
    2. In the file, replace the parameters by something like this:
    ## Assign variables
rg=AzureTipsAndTricksGroup
location=eastus2

    # Conclusion

    The Azure Kung Fu Toolkit contains lots of practical Azure CLI scripts and examples that you can easily use by replacing parameters and running the scripts. Go and check it out!
