                                                                                                # Share Business Logic between Azure Functions

                                                                                                A common scenario to share business logic between function is: I have an Azure Function that runs daily on a timer trigger and need the same function to run when I request it manually. While the template used may change you basically donâ€™t want to copy/paste the logic into another function. Instead, your function app architecture could look like:

|     host.json
|____ shared
|     |____ businessLogic.js
|____ function1
|     |____ index.js
|     |____ function.json
|____ function2
      |____ index.js
      |____ function.json

                                                                                                In the shared folder, you'd create a file named businessLogic.js and put your shared code.

                                                                                                A sample in JS could be the following:

                                                                                                module.exports = function (context, req) {
    context.res = {
        body: "<b>Hello World, from Azure Tips and Tricks</b>",
        status: 201,
        headers: {
            'content-type': "text/html"
        }
    };
    context.done();
};

                                                                                                var logic = require("../shared/businessLogic.js");
module.exports = logic;
